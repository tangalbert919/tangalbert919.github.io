What is MyZ Reborn?

MyZ Reborn is an attempt at reviving an old plugin called MyZ. MyZ is a plugin that allows players to have that "post-apocalyptic" feel while surviving against the undead in more realistic ways (in other words, take care of your thirst, hunger, and don't get infected). It's not a huge rewrite of the plugin, but it's a huge upgrade so it works with recent versions of Minecraft.

If you want updated versions of MyZ, you will have to come to me. I am working on getting the really good parts of the plugin back, but right now, the plugin isn't 100% working at the moment. I need help with fixing a lot of errors found on runtime so we can all have fun on zombie apocalypse servers again. The source code is found on GitHub, and I did provide a link to it. If you want to contribute to the code, please do. Just make sure you know Java, and are able to work with the CraftBukkit or Spigot class files. If you want to learn more about MyZ, click the "Bukkit" image.
